Below is a comprehensive list of common Git commands:

Initialization and Configuration:

git init: Initialize a new Git repository in the current directory.
git config: Configure Git settings (e.g., user name, email, etc.).
Basic Commands:

git add: Add changes to the staging area.
git commit: Commit the changes from the staging area to the repository.
git status: Show the status of the working tree and staging area.
git diff: Show the differences between the working tree and the staging area.
Branching and Merging:

git branch: List, create, or delete branches.
git checkout: Switch to a different branch.
git merge: Merge changes from one branch into the current branch.
Working with Remotes:

git remote: List, add, or remove remote repositories.
git clone: Clone a remote repository to a local directory.
git pull: Fetch and merge changes from a remote repository.
git push: Push local commits to a remote repository.
Commit History and Log:

git log: Show commit history.
git show: Show information about a specific commit.
Undoing Changes:

git revert: Create a new commit that undoes a previous commit.
git reset: Unstage changes or move the HEAD to a previous commit.
git stash: Temporarily save changes to work on something else.
Tags:

git tag: List, create, delete, or verify tags.
Remote Branches:

git fetch: Download objects and refs from another repository.
git remote show: Show information about a remote.
Advanced:

git rebase: Reapply commits on top of another base commit.
git cherry-pick: Apply changes from a specific commit to the current branch.
git bisect: Find which commit introduced a bug using binary search.
Git Submodules:

git submodule: Add, update, or remove submodules from a repository.
Git Ignore:
.gitignore: A file used to specify which files or directories to exclude from version control.
These are some of the most commonly used Git commands. You can get more information about each command using git help <command> or git <command> --help. Additionally, you can find extensive documentation on Git's official website: https://git-scm.com/doc
